# Research Journal — Overview

**Purpose:** Transparent documentation of research process following German Wissenschaftsethik (science ethics)

## Structure

```
research_journal/
├── sessions/           # Detailed session logs
│   └── YYYY-MM-DD-session-XX.md
├── decisions/          # Major methodological/theoretical choices
│   └── project_evolution.md
└── methodology/        # Research ethics and methods
    └── ethics_and_methodology.md
```

## Standards

**Each session log includes:**
1. Initial situation (Ausgangslage)
2. Actions undertaken
3. Decisions made (with rationale)
4. Problems encountered and solutions
5. Results/outputs
6. Open questions
7. Next steps
8. Methodological reflection

**Decision logs track:**
- Scope changes
- Theoretical pivots
- Methodological innovations
- Error corrections

## Principles (Nachvollziehbarkeit)

1. **Verifizierbarkeit** — All claims verifiable through Git history
2. **Korrekturoffenheit** — Errors acknowledged, not hidden
3. **Begründungspflicht** — Decisions explained, not arbitrary
4. **Quellentransparenz** — Sources traced, evaluated

## Relation to Other Project Files

- **literature/overview.md** — What we read
- **research_journal/** — How and why we read it
- **notes/concepts/** — What we think about it
- **chapters/** — What we write from it
